# 基于瑞芯微RK3588的甘蔗地智慧农业云端作业系统

本项目为第9届“AI赋能设计，设计点亮AI”参赛作品。针对南方甘蔗生长中易受草害影响、传统除草效率低及药液浪费严重等痛点，设计并实现了一套基于瑞芯微 RK3588 的甘蔗地数字化建图与靶向喷药作业系统。

## 🛠️ 软件与系统架构
本系统采用 **RK3588 + STM32** 双处理器架构，形成了“边缘计算 + 本地服务器”的端边协同闭环：
* **RK3588 边缘智算主控**：运行 ROS2 (Humble/Galactic)，利用 NPU 部署 YOLOv8/RKNN 模型，实现作物与杂草的 AI 智能识别；融合激光雷达点云与底盘里程计，通过 Cartographer 算法完成实时二维语义建图，负责全局任务调度与数据管理。
* **STM32 底层控制板**：基于 FreeRTOS 实时操作系统，通过高速 DMA 串口网桥接收 RK3588 核心指令，精确控制履带底盘闭环运动、喷药机械臂姿态调整、继电器水泵靶向喷药以及土壤采样机构。
* **PC端 Web 地面站**：基于 Flask + SocketIO 架构，提供端边数据中转、实时视频推流监控、语义地图标绘显示及多媒体历史资产数字化自动归档。

## 📂 项目主要代码结构
由于 ROS2 项目高内聚特性，核心业务代码集中于本仓库的功能包中：
* `main.py`：RK3588 边缘智算中心总控程序，联动推理线程池、ROS2 订阅、串口调度及任务报告生成。
* `robot_node.py`：STM32 串口数据网桥模块，滑动窗口查找 AA55 里程计帧与 BB2A 土壤帧并分流发布 ROS2 话题。
* `aotu.py`：全自动巡检独立状态机，管理待机(IDLE)、直行(TRACKING)、调头(TURNING)与收车归档(FINISHED)状态。
* `filter.py`：激光雷达点云裁剪过滤脚本，滤除车体自身无效遮挡区域。
* `media_uploader.py`：任务资产 TCP 分块同步模块，实现多媒体资产至 PC 地面站自动归档。
* `brain_report.py`：本地 AI 专家报告生成模块，调用本地大模型生成面向农户的养护建议。

## 📊 主要性能指标
* **边缘视觉推理**：RKNN YOLOv8，输入分辨率 640×480，支持甘蔗、杂草和病害类识别。
* **雷达建图链路**：Cartographer 二维建图，盲区 0.15m，默认点云范围 4m。
* **轨迹记录周期**：任务运行且运动时，约 0.2s 记录更新一次行驶轨迹。
* **喷药控制接口**：支持左右方向定向喷施，时长 1-5s 可调，当前采用 25cm 标定参考距离。
* **自动巡检参数**：默认行长 6m、行距 0.70m、总行数 4 行，支持 Web 端参数修改。
